# Проект: Место

### Описание проекта

Проект представляет собой интерактивную страницу, куда можно добавлять фотографии, удалять их и ставить лайки.

**При написания проекта использовались следующие технологии:**
* HTML
* CSS
* Figma
* JavaScript
* React
